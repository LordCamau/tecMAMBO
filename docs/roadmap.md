# tecMAMBO roadmap

## Phase 3, explicitly not built in this pass

- Liveblog support with `LiveBlogPosting`
- Deep author E-E-A-T and methodology pages
- Multiple newsletter product lines
- Reading extras: progress bar, save-for-later, and text-to-speech
- Compare Phones as a richer product surface in the main IA
- i18n and hreflang
- PWA and push notifications
- Dedicated video lane
- Read-only MCP endpoint

The data model added in v2 does not block these surfaces. Compare Phones remains available at `/compare-phones`, but it is not in the v2 top navigation.
