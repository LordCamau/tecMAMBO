# Regional Layer

tecMAMBO stays global-first. Region is a cross-cutting taxonomy on articles, not a competing top-level editorial section.

## Decision

African technology coverage is collected through:

- `/africa`
- `/africa/[country]`
- region chips on articles and cards
- one homepage lane named `Tech across Africa`
- a footer link and a `More` menu link

We are not adding a top-level `Africa` nav slot yet. The current path gives regional stories strong internal linking while keeping News, Reviews, Wallet Watch, Business, and other global formats primary. Revisit a top-level slot once regional coverage is consistently deep across multiple countries.

## Editorial Rule

Tag a story with a country Region when the story is about that country or especially useful to readers there. Use more than one Region only when the story truly spans those countries. Leave global stories untagged.

The taxonomy is designed to extend beyond Africa later, so tecMAMBO can build other regional branches without changing article URLs.
