# tecMAMBO navigation v2 decisions

Date: 2026-06-26

## Signature formats outside the main nav

The v2 nav was implemented as provided. Explainers, Real Life, and Opinion remain first-class sections, keep their format badges, appear in the footer discovery links, are filterable from `/latest`, and have dedicated homepage lanes. Recommended follow-up: consider returning Explainers to the top-level nav because it carries half of the brand promise with Glossary.

## Computing vs Laptops & PCs

Implemented as one canonical topic: `computing`. The label changes by context: News uses “Computing”, Reviews and Wallet Watch use “Laptops & PCs”.

## Business

Implemented as a top-level section and editorial format at `/business`, with `Startups` as its reserved topic archive at `/business/startups`.

## Compare Phones

The tool remains built and routable at `/compare-phones`, but it is removed from the main nav per v2. It is documented as a Phase 3 product surface and linked from footer discovery for now.
